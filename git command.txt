git init
    지역 저장소 생성

git config user.name
    프로젝트별 지역 사용자 등록    
    ex) git config khs779

git config user.email
    프로젝트별 지역 사용자 등록
    ex) git config khs779@gmail.com

git config --global user.name    
    지역환경의 전체프로젝트를 위한 사용자 등록

git config --global user.email
    지역환경의 전체프로젝트를 위한 사용자 등록

git remote add
    원격 저장소의 주소를 지역 저장소에 등록
    ex) git remote add https://github.com/kim779/study.git  

gid add
     git add -A //바뀐거 전부 스테이징
    커밋에 포함될 파일 등록

git status
    현재 프로젝트의 파일 상태확인

git branch -M  test
    브래치를 만든다

git commit
    새로운 커밋 생성

git commit --amend
    기존 커밋 수정

git logg
    커밋내용확인
git log --pretty=oneline --graph

git push "원격저장소식별자" "브랜치"
    ex) git push origin branch1

git clone


git branch -a
	브랜치 정보 보기

git branch -d [브랜치명]
	지역저장소 브랜치 삭제하기

git branch origin -d [브랜치명]
	원격저장소 브랜치 삭제하기

git checkout main
    main 브랜치를 작업브랜치로 만들었다

git merge otherbranch
    다른브랜치를(otherbranch)를 작업 브래치와 병합한다
